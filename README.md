`ethereum_abi` is a Rust library to help writing code that interacts with Ethereum Smart Contracts.

## Examples

See the [examples](examples) folder.

## Features

### ABI encoder V1

- [x] JSON parsing
- [x] Function selectors (method ID)
- [x] argument encoding and decoding

### ABI encoder V2

- [x] JSON parsing
- [x] Function selectors (method ID)
- [x] argument encoding and decoding
